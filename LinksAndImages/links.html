This example is about links and images.

This page is about links. There are three kinds of links:

  • Absolute Links
    • The most detailed way to refer to a web resource, like to Mozilla Developer Network , which is a good resource.
  • Relative Links
    • These point to a file in the same website and the same directory as the current file. The scheme and domain are implied to be the same. Like to the Extras page.
  • Root-Relative Links
    • These links are relative to the root URL of the website. This does mean that without a server these can't be used.
  • It is also possible to get to parent folders by using the .. syntax. This can be seen in action in the Extras page.
